# Projector 0.0.1

Release Date: 2026-04-19

## Introduction

Projector is a personal project statistics tool used to count the number, types, modification times, and other information of personal projects, helping developers better understand the status of their projects.

Projector is built with `clap` for command-line parsing and `serde` for serialization and deserialization, providing concise and powerful project statistics functionality.

## Features

- **`list` command**: Counts the number, types, modification times, and other information of personal projects, automatically analyzes and outputs detailed reports
- Supports multiple project type recognition
- Sorts by modification time to quickly understand recently active projects
- Clean and beautiful command-line output format

## Installation

```bash
# Install from source
git clone https://github.com/hunter-hongg/projector.git
cd projector
cargo build --release

# Or install directly with cargo
cargo install --path .
```

## Usage Examples

```bash
# Count all projects in the current directory
projector list

# Count projects in a specified directory
projector list /path/to/projects
```

## Tech Stack

- Rust programming language
- clap: Command-line argument parsing
- serde: Serialization and deserialization
- chrono: Time handling
